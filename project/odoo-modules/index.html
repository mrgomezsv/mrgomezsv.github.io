Módulo Principal DTE Base
mrg_dte_sv

Facturación Electrónica DTE El Salvador

El corazón de la facturación electrónica. Permite generar, firmar digitalmente y transmitir todos los Documentos Tributarios Electrónicos (DTE) requeridos por el Ministerio de Hacienda de El Salvador de forma nativa desde las facturas de Odoo.

Se integra con:
  • Odoo Invoicing & Accounting (Facturación y Contabilidad)
  • Odoo Contacts / Customers (Clientes y Proveedores)
  • API Services / MiFacturaDTE (Transmisión segura)
Generación de Facturas, Crédito Fiscal, Notas de Crédito, Notas de Débito, etc.
Firma electrónica y validación del Ministerio de Hacienda en tiempo real.
Flujo inteligente de gestión de Contingencias (Offline y eventos especiales).
Generación del PDF y Ticket de Representación Gráfica homologados.
Fiscal Declaraciones
mrg_dte_sv_books

Libros y Anexos Fiscales DTE

Automatiza la contabilidad fiscal salvadoreña de IVA. Este módulo toma todos los DTEs emitidos y recibidos para generar automáticamente los libros legales y los anexos de la declaración de IVA mensuales.

Se integra con:
  • Módulo Base mrg_dte_sv
  • Odoo Localizaciones & Contabilidad Estándar
Libro de Compras, Ventas a Consumidores Finales y Ventas a Contribuyentes.
Exportación directa a formato CSV y XLSX compatible con el validador del Ministerio de Hacienda (F07).
Filtros avanzados y reportes resumidos en PDF para auditorías rápidas.
Contabilidad Reporting
mrg_dte_sv_dynamic_reports

Informes Financieros Dinámicos

Permite a los contadores y administradores generar balances y estados financieros personalizables y dinámicos adaptados a la moneda local (USD) y al formato oficial de presentación exigido en El Salvador.

Se integra con:
  • Plan de Cuentas Contables (Chart of Accounts) de Odoo
  • Odoo Localización Salvadoreña
Visualización dinámica e interactiva interactuando directamente en Odoo.
Balance de Comprobación, Balance General, Estado de Resultados y Libro Diario.
Capacidad de desglose (drill-down) interactivo para rastrear el origen de cada monto.
Punto de Venta Retail
mrg_dte_sv_pos

DTE El Salvador - Punto de Venta (POS)

Optimizado para entornos de alta transaccionalidad. Permite emitir Facturas de Consumidor Final (FCF) y Créditos Fiscales (CCF) electrónicos directamente en la pantalla de ventas de Odoo POS con solo un clic.

Se integra con:
  • Odoo Point of Sale (POS) Core
  • Módulo Base de Facturación Electrónica mrg_dte_sv
Envío automático del DTE a Hacienda al momento de pagar la orden en el POS.
Soporte completo para tiqueteras térmicas de 80mm e impresión del código QR oficial de consulta.
Gestión de datos de facturación especial de clientes directamente desde la pantalla táctil.
Seguridad Auditoría
mrg_dte_sv_pos_security

DTE POS Security SV

La seguridad es prioridad en el punto de venta. Este módulo añade capas adicionales de validación, auditoría y control de permisos en las operaciones críticas de DTE dentro de Odoo POS para evitar fraudes o errores humanos.

Se integra con:
  • Odoo POS y el módulo mrg_dte_sv_pos
  • Odoo User & Group Permissions (Gestión de usuarios estándar)
Restricción y control de permisos para la anulación de DTEs o notas de crédito en POS.
Bitácora de auditoría detallada que registra quién, cuándo y por qué aprobó una operación DTE.
Control estricto de arqueo de caja cruzando DTEs válidos, anulados e inválidos.

Recepción & Sello de Recepción

1. Generación Local

Odoo genera el esquema JSON estructurado según las especificaciones técnicas vigentes de Hacienda a nivel de base de datos.

2. Firma y Envío Seguro

El microservicio API procesa el DTE, aplica la firma digital con el certificado del cliente y lo envía instantáneamente a Hacienda.

3. Autorización Inmediata

El Ministerio de Hacienda procesa, valida el DTE y devuelve el Sello de Recepción oficial, el cual Odoo almacena y vincula a la factura.
