📋 Novedades v1.0.2 Beta

Esta versión incluye correcciones importantes y optimizaciones de rendimiento para una experiencia más fluida.

v1.0.2 Beta

🎀 Kitty Runner

📅 Mayo 2026

  • Fix Login seguro: Corregido crash al iniciar sesión cuando el usuario es null.
  • Fix UI reactiva: Eliminada latencia de Firestore en actualización de pasos. Los datos se reflejan al instante.
  • Fix Data loss: Al desactivar una app ya no se pierde su configuración personalizada.
  • Fix setState post-async: Agregadas verificaciones mounted en profile_view para evitar errores.
  • Fix StreamBuilder: Extraído WelcomeHeader a StatefulWidget para evitar recreaciones infinitas.
  • Perf Colores centralizados: Todos los colores de la app ahora se definen en AppTheme. Consistencia visual y fácil mantenimiento.
  • Perf Debounce nativo: Llamadas al canal de bloqueo Android se debouncean 200ms para evitar saturación.
  • Perf Logger optimizado: Buffer de logs limitado a 200 entradas sin copias innecesarias en memoria.
  • Perf Blur reducido: BackdropFilter de 20px → 8px, menos carga en GPU sin perder el efecto.
  • Perf Achievements cacheados: 0 lecturas a Firestore por logro ya verificado en la sesión.
  • Perf Widget liviano: Eliminado GestureDetector redundante en CupertinoSwitch.
  • Perf Logging: Todos los print() reemplazados por LoggerService.log con sistema de logging centralizado.
